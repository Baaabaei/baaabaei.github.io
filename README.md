Production-ready personal portfolio for an AI Engineer

Core Design Principles

1. Recruiter-First UX
   · Information hierarchy prioritizes key skills and projects.
   · Clear calls-to-action in the hero section.
   · Project cards follow a "Problem → Solution → Result" narrative.
2. Performance & Accessibility
   · Fully static HTML/CSS/JS — no build tools or dependencies required.
   · Responsive layout that works on all screen sizes.
   · Semantic HTML for better screen reader compatibility.
3. Visual Identity
   · Gradient (blue to purple) used for the brand color.
   · Subtle hover animations to enhance interactivity without being distracting.
   · Clean typography using the Inter font.

Project Showcase

· Vision Transformer for Hoda: Shows deep learning for computer vision.
